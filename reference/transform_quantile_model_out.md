# Transform quantile model output into a forecast object

Transform quantile model output into a forecast object

## Usage

``` r
transform_quantile_model_out(model_out_tbl, oracle_output)
```

## Arguments

- model_out_tbl:

  Model output tibble with predictions

- oracle_output:

  Predictions that would have been generated by an oracle model that
  knew the observed target data values in advance

## Value

forecast_quantile
